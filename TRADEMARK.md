# Linmas Trademark and Brand Policy

The Linmas name and branding are not licensed under the Apache-2.0 software license.

## Allowed

You may:
- refer to Linmas factually as the original upstream project
- state that your project or fork is based on Linmas
- compare your work with Linmas in documentation or commentary

## Not Allowed Without Permission

You may not:
- market a fork or derivative as if it were the official Linmas project
- use the Linmas name in a misleading product or package title
- imply endorsement, partnership, certification, or official status by the Linmas project
- publish paid or free derivative branding such as "Linmas Pro", "Linmas Enterprise", or similar names that create confusion about origin or affiliation
- use Linmas logos, visual branding, or official-looking identity assets in a way that suggests your distribution is official

## Recommended Naming for Forks

If you fork or adapt Linmas, use a distinct project name and clearly state that it is an independent derivative based on Linmas.

Example:
- "Acme Security Skills, based on Linmas"

Not recommended:
- "Linmas Pro"
- "Official Linmas Enterprise"
- "Linmas Premium"

## Permission Requests

If you want to use Linmas branding beyond the factual references allowed above, request written permission from the project owner.
